# Algoritmos e Estruturas de Dados 1 - PUC
Repositório da disciplina **Algoritmos e Estruturas de Dados 1** do primeiro período de Engenharia de Software da PUC!

Aqui estão reunidas todas as listas de exercícios resolvidas durante o semestre, organizadas por temas e tópicos. O objetivo deste repositório é facilitar o acompanhamento dos conteúdos, a revisão dos conceitos e a prática de programação em C.

## 📁 Estrutura do Repositório

- **lista00/**: Exercícios introdutórios e complementares.
- **lista1-condicional/**: Estruturas condicionais.
- **lista2-condicional/**: Exercícios adicionais de condicionais.
- **lista3-while-dowhile/**: Laços de repetição (`while`, `do-while`).
- **lista4-funcao/**: Funções em C.
- **lista5-funcao-e-ponteiros/**: Funções e ponteiros.
- **lista6/**: Exercícios avançados, vetores, strings e registros.
- **listas-avaliativas/**: Listas avaliativas da disciplina.
- **verde/**: Materiais extras e desafios.

## Sobre a Disciplina

A disciplina aborda lógica de programação, estruturas de controle, funções, ponteiros, vetores, strings e registros, fundamentais para a formação em Engenharia de Software.

## Como Executar

1. Navegue até a pasta desejada.
2. Compile o arquivo `.c` com um compilador C, por exemplo:
   ```sh
   gcc lista1-condicional/ex1.c -o ex1
   ./ex1
   ```
3. Siga as instruções do exercício no terminal.
